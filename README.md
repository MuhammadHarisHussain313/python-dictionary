# Python Dictionary Practice

This repository contains basic to intermediate Python dictionary operations.

###  Topics Covered

- Creating dictionaries
- Accessing values
- Adding key-value pairs
- Updating values
- Removing elements (`del`, `pop`)
- Looping through keys, values, and items
- Checking key existence
- Dictionary methods:
  - `get()`
  - `update()`
  - `pop()`
  - `clear()`
  - `copy()`
  - `setdefault()`
- Length of dictionary
